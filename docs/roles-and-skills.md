## Roles and responsibilities

- **Back-end engineer** – writes server logic, API, and works with databases.  
- **Front-end engineer** – creates the user interface and ensures it works.  
- **QA engineer** – tests features and reports bugs.  
- **Data engineer** – processes and manages logistics-related data.  
- **DevOps engineer** – deploys services and monitors servers.


## Common skills across roles

- Programming: Python, Java, JavaScript  
- Databases: PostgreSQL, MySQL  
- Testing: Unit tests, basic automation  
- Deployment: Docker, Git  
- Cloud basics: AWS
